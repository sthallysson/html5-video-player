# HTML5 Video Player

This project is a custom video player made with vanilla javascript.

## Stack utilizada

- HTML5
- Javascript
- TailwindCSS

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/sthallysson/html5-video-player
```

Entre no diretório do projeto

```bash
  cd html5-video-player
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
